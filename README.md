## 🧰 Tools

### 📊 Campaign Builder
> Plan campaigns, manage assets, and generate content — all in one place.

An open-source campaign framework builder for B2B and SaaS marketing teams. Define your messaging, organize assets by funnel stage, map buyer journeys, and use AI to generate campaign copy grounded in your actual framework.

| | |
|---|---|
| **Status** | ✅ Live & Published |
| **Tech** | Node.js · Express · Vanilla JS · Anthropic API |
| **License** | MIT |
| **Docs** | [`campaign-builder-public/`](campaign-builder-public/README.md) |

**Highlights:**
- 🎯 Per-product personas and channel management — no JSON editing required
- 📦 Asset repository with search, tagging, status tracking, and CSV export
- ✨ AI content generator for 11 content types (emails, landing pages, blog posts, and more)
- 🗺️ Visual buyer journey mapping with drag-and-drop canvas
- 📅 Quarter planning grid — 13 weeks × 4 funnel stages
- 🔄 Editable campaign flow touchpoints

## 🚀 Philosophy

Most marketing software is either too expensive, too bloated, or built for ops teams — not marketers. babel-system is different:

- **Open source** — fork it, customize it, make it yours
- **Lightweight** — no databases, no complex setup, runs locally in minutes
- **AI-native** — built around LLMs from the start, not bolted on
- **Practitioner-first** — designed by a marketer, for marketers
