Getting Started

These are demo products. To use your own:

  1. Edit data/campaign-frameworks.json
  2. Add your product name, tagline, and 4 pillars
  3. Start adding assets and building your journey
  4. See CUSTOMIZATION.md for full details
🎯

Personas & Channels

Both are managed inside the app — no JSON editing needed.

  1. Select a product above
  2. Go to the Framework tab
  3. Click ✏️ Edit Framework
  4. Edit Target Personas and Channels to match your go-to-market

These appear as tagging options on every asset and in the Content Generator.

✨

AI Content Generation

Generate copy using the Anthropic API — free to start, 2-minute setup.

  1. Get a key at console.anthropic.com
  2. Copy .env.example → .env
  3. Set ANTHROPIC_API_KEY=sk-ant-...
  4. Restart: npm start
